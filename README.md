### Installation

Use [Composer](https://getcomposer.org) to install the library.

```
    $ composer require yakeing/badge
```

### Initialization parameter

- [x] Sample：
```php
    $Badge = new Badge();
    $label = 'label';
    $message = 'message';
    $color = '##4c1';
    $Badge->OutCode($label, $message, $color);
```
